PETUNJUK

1. Pilih menu "Identifikasi Sleep Apnea" pada sidebar
2. Klik tombol "Choose File", kemudian pilih file EEG dalam format .csv
3. Klik tombol "Proses csv", kemudian tunggu hingga proses identifikasi selesai
4. Hasil identifikasi Sleep Apnea akan muncul pada tempat yang telah ditentukan
5. Jika file tidak sesuai syarat dapat lakukan ulang proses nomor 2
6. Jika ingin membersihkan hasil Identifikasi, dapat menekan tombol "Reload"
7. Tombol "Petunjuk" pada sidebar digunakan untuk masuk pada halaman petunjuk
8. Tombol "Beranda" pada sidebar digunakan untuk masuk pada halaman beranda
